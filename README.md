# 📊 Student Lifestyle Data Visualization

## 🔍 Overview

This project focuses on analyzing and visualizing a **student lifestyle dataset** to understand how different factors like sleep, study habits, and daily routines impact academic performance.

The goal is to extract meaningful insights using **data visualization techniques**.

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* Matplotlib
* Seaborn

---

## 📁 Dataset

The dataset used:

* `student_lifestyle_dataset.csv`

It includes features like:

* Sleep hours
* Study patterns
* Exam scores
* Other lifestyle-related attributes

---

## ⚙️ Steps Performed

### 1. Data Loading

* Imported dataset using Pandas
* Displayed initial rows for understanding

### 2. Data Inspection

* Checked data types and structure (`info()`)
* Explored column names

### 3. Data Cleaning

* Standardized column names:

  * Lowercase conversion
  * Removed spaces
  * Replaced spaces with underscores

---

## 📊 Visualizations

### 🔹 Histogram

* Displayed distribution of all numerical features
* Helped identify spread and skewness

### 🔹 Correlation Heatmap

* Visualized relationships between variables
* Identified strong positive/negative correlations

### 🔹 Scatter Plot

* Compared **sleep hours vs exam score**
* Showed how sleep affects performance

### 🔹 Pairplot

* Explored relationships across multiple variables
* Provided a complete overview of feature interactions

---

## 📈 Key Insights

* Sleep patterns show a noticeable relationship with exam scores
* Some lifestyle factors strongly correlate with performance
* Visualization helps uncover trends not obvious from raw data
